Code:

(get the maximum of the stack)
duplicate 2 2 size / -
for {
    max
}

(get the minimum of the stack)
duplicate 2 2 size / -
for {
    min
}

(compare them)
==

Factorial

Description: Get the factorial of N

Code:

(generate a list like:
    1 2 ... N N)
N 0 swap for {
    1 + dup
}

(turn the list into
    1 2 ... N [N-1])
1 swap -

(multiply to get the factorial)
for {
    *
}
